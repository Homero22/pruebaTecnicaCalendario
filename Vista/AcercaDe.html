Acerca De

La página inspirada en los conocimientos brindados en clase es un CRUD, acrónimo de Create, Read, Update and Delete, (Crear, Leer, Actualizar y Borrar), el servicio que ofrece el sistema en primer lugar tiene un acceso mediante usuario y contraseña, al llenarlo este va a poder acceder a un menú de opciones en donde podrá realizar las acciones ya mencionadas, el ingreso de datos con un registro de estos en la base de datos previamente creada con MySQL además de tener, la opción de registrar una compra ingresando todos los datos que se le pide en los campos que despliega un formulario fácil y comprensible para la calidad del servicio brindado en la web, seguido de ello podrá modificar y eliminar datos permitiendo así no registrar la compra este registro se guarda en el caso de ser modificado, para el caso de ser eliminado el usuario eliminara todos los datos que se hayan registrado, una de las funcionalidades importantes que también se requiere es una búsqueda y una muestra de los datos buscados donde se buscara el registro que el usuario haya realizado, en el caso de que el usuario no lleno ningún formulario con información, se despliega un menú para ingresar datos si así lo desea, por otro lado en el caso de ser encontrado se le desplegara la información que la base de datos tenga sobre ese registro llenado previamente. Por otro lado el usuario tendrá el acceso a registros de ventas, clientes, productos, etc. Esto para tener resumen de todas y cada una de las compras y clientes registradas, de esta manera se podrá tener la información a la mano además del registro de facturación y así monitorear la economía de la juguetería.
